# Website Company - Premium Landing Page

**Website ini sepenuhnya buatan W for Wijaya.**

Website company profile statis ini didesain dengan tingkat estetika tinggi, menggunakan elemen modern seperti glassmorphism, animasi mulus (Framer Motion), typography eksklusif, serta responsivitas sempurna di seluruh perangkat (Desktop, Tablet, dan Mobile). 

## 👨‍💻 Dibuat Oleh
**W for Wijaya**

## Fitur Utama
- **Desain Premium:** Gelap, elegan, dan menawan dengan aksen *amber*.
- **Animasi Mulus:** Menggunakan Lenis untuk *smooth scrolling* dan Framer Motion untuk interaksi elemen.
- **Responsif:** Tampil memukau di perangkat seluler dan desktop.
- **Performa Tinggi:** Dibangun di atas React, TypeScript, dan Vite.

---

## 🚀 Cara Menjalankan Secara Lokal

1. Instal dependensi:
   ```bash
   npm install
   ```
2. Jalankan development server:
   ```bash
   npm run dev
   ```
